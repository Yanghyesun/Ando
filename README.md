## Git Convention

> **Cycle**

- 하루에 한번 merge.



> **Branch**

- master
  - <kbd>develop</kbd>
    - <kbd>Main</kbd>
    - <kbd>Crime</kbd>
    - <kbd>Safety</kbd>


> **Commit Message**

```
feat - new feature
update - When implementing a function
fix - When bugs was discovered
```

*ex) `[gitmoji] update : [Jira Issue number] Home.vue Timer Function Updated`*
